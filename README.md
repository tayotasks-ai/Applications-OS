# Agency Management OS - Applications Frontend Portal

A premium Vue 3 + Tailwind CSS + Pinia SaaS educational travel management application frontend, built for high-performance CRM admissions dispatching and multi-tenant scaling.

## Features
- **SaaS B2B Multi-Tenancy Architecture**: Completely isolated organizational views under strict workspace parameters.
- **Applications Master Kanban Board**: Gated, high-security pipeline board for Team Leads (`OfficerLead`) featuring active staff workload filters.
- **Internal journey Timelines**: Visual glowing steppers integrated into Counselor detailed views mapping admissions program submissions.
- **Dynamic Pricing Guardrails**: Auto-enforcement of Seat boundaries, candidate pipelines limits, and secure document storage ceilings (Free vs. Growth vs. Enterprise).

## Technology Stack
- **Core Framework**: Vue 3 (Composition API)
- **Routing**: Vue Router
- **State Store**: Pinia
- **Styling Engine**: Tailwind CSS v4 & PostCSS
- **Bundler & Tooling**: Vite

## Project Startup

### Installation
```bash
npm install
```

### Run Locally (Development)
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
